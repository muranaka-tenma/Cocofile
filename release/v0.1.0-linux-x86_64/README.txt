CocoFile v0.1.0 Alpha - Linux版

## インストール方法

1. 両方のファイルを同じフォルダに配置してください：
   - cocofile (メインアプリケーション)
   - python-analyzer-x86_64-unknown-linux-gnu (分析エンジン)

2. 実行権限を確認：
   chmod +x cocofile
   chmod +x python-analyzer-x86_64-unknown-linux-gnu

3. アプリケーションを起動：
   ./cocofile

## 必要要件

- Linux x86_64
- glibc 2.31以降
- X11サーバー（WSL2の場合）
