### 1. GUIテスト実施（手動作業）⭐ 最優先
**作業内容**: Windows/macOS/Linuxデスクトップ環境での実際のアプリ操作テスト

**必要な人**:
- Windows/macOS/Linux環境を持つ人（デスクトップ環境必須）
- Node.js + Rustがインストール済み、またはインストール可能な人

**手順書**:
- 📘 詳細版: `docs/GUI_TEST_GUIDE.md` (46項目、30-45分)
- 📋 簡易版: `docs/GUI_TEST_CHECKLIST.md` (29項目、10-15分)

**実施方法**:
```bash
# Windows/macOS/Linuxのターミナルで
cd /path/to/CocoFile/frontend
npm run tauri:dev
# → デスクトップウィンドウが開く
# → 手順書に従ってマウス・キーボードで操作
```

**重要**: これは**人間が手動で実施する作業**です。エージェントでは実施できません。
