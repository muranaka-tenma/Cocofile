# スキャン・インデックス管理画面（S-003）E2Eテスト仕様書

生成日: 2025-11-04
対象画面: ScanIndexScreen
総テスト項目数: 28項目

## テスト項目一覧

### 正常系（12項目）

1. 画面初期表示 - 統計情報が表示される
2. 手動スキャン開始 - スキャンが開始される
3. スキャン進捗表示 - リアルタイムに更新される
4. スキャン完了 - 完了メッセージが表示される
5. スキャン停止 - 進行中のスキャンを停止できる
6. インデックス統計表示 - ファイル数、サイズ、日時が表示される
7. 重複ファイル検出 - 重複グループが表示される
8. 重複ファイル詳細 - パスとサイズが表示される
9. データベースクリーンアップ - 削除件数が表示される
10. インデックス再構築 - 再構築が開始される
11. 進捗率表示 - パーセンテージが正しく表示される
12. 残り時間推定 - 推定時間が表示される

### 異常系（8項目）

13. スキャンエラー - エラーメッセージが表示される
14. 重複スキャン防止 - 実行中は開始できない
15. 空のフォルダ - 適切に処理される
16. アクセス権限エラー - エラーが表示される
17. データベースエラー - リトライ可能
18. ディスク容量不足 - 警告が表示される
19. ネットワークエラー - 適切にハンドリング
20. タイムアウト - 長時間スキャンの処理

### パフォーマンス（5項目）

21. 大量ファイル処理 - 10万ファイルでも動作
22. メモリ使用量 - 500MB以下を維持
23. CPU使用率 - 50%以下を維持
24. スキャン速度 - 100ファイル/10秒
25. UI応答性 - スキャン中もUIが操作可能

### UI/UX（3項目）

26. ローディング状態 - 適切に表示される
27. 確認ダイアログ - 破壊的操作前に確認
28. レスポンシブ - モバイルでも表示可能
