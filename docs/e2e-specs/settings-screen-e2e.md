対象画面: SettingsScreen

## テスト項目一覧

### 正常系（15項目）

1. 画面初期表示 - 設定データが正しく表示される
2. 監視フォルダ追加 - 新しいフォルダが追加される
3. 監視フォルダ削除 - フォルダが削除される
4. スキャンタイミング変更 - リアルタイム/アイドル/手動が切り替わる
5. 自動収納設定変更 - ON/OFFが切り替わる
6. デフォルトタグ追加 - 新しいタグが追加される
7. デフォルトタグ削除 - タグが削除される
8. 除外フォルダ追加 - 除外リストに追加される
9. 除外フォルダ削除 - 除外リストから削除される
10. 除外拡張子追加 - 拡張子が追加される
11. 除外拡張子削除 - 拡張子が削除される
12. 設定の保存 - 変更が永続化される
13. 設定のリセット - デフォルト値に戻る
14. バリデーション - 無効なパスが拒否される
15. 重複防止 - 同じフォルダ/タグは追加できない

### 異常系（8項目）

16. API エラー - エラーメッセージが表示される
17. 空の入力 - 空白では追加できない
18. 無効なパス - 存在しないパスは拒否される
19. ネットワークエラー - リトライ可能
20. タイムアウト - 適切にハンドリングされる
21. 最大数制限 - 上限に達すると警告表示
22. 無効な拡張子 - 正しい形式が要求される
23. 権限エラー - アクセス権がない場合の処理

### UI/UX（7項目）

24. ローディング状態 - スピナーが表示される
25. 成功通知 - 保存成功時にトースト表示
26. エラー通知 - エラー時にメッセージ表示
27. 変更検出 - 未保存の変更が検出される
28. 確認ダイアログ - 削除時に確認が表示される
29. キーボード操作 - Enterキーで追加できる
30. レスポンシブ - 様々な画面サイズで正しく表示
