## 🔮 Phase 9以降: 将来の機能拡張

### Phase 9-A: 高度なファイル整理機能

**推定時間**: 5-8時間

#### 機能詳細

1. **ユーザー定義ルールのUI実装**
   - ルール作成画面
   - 条件設定（ファイル名パターン、タイプ、サイズ等）
   - 優先度設定
   - 有効/無効切り替え

2. **移動履歴のUI実装**
   - 履歴一覧表示
   - フィルター機能
   - アンドゥ機能（移動の取り消し）

3. **学習ベースの提案**
   - ユーザーの移動パターンを学習
   - 提案精度の向上
   - 信頼度の動的調整

4. **高度なクラウド同期検出**
   - Windows API統合
   - オンラインのみファイルの検出
   - 同期中ファイルの警告表示

---

### Phase 9-B: Python解析エンジン統合

**推定時間**: 3-5時間

#### 機能詳細

1. **PyInstallerでバイナリ化**
   - Python分析エンジンを単一実行ファイル化
   - Tauriアプリに同梱
   - 依存関係の削減

2. **統合テスト**
   - PDF/Excel/Word/PowerPoint解析のテスト
   - パフォーマンステスト
   - エラーハンドリング確認

---

### Phase 9-C: パフォーマンス最適化

**推定時間**: 2-4時間

#### 最適化項目

1. **データベースクエリ最適化**
   - FTS5検索の高速化
   - インデックスの追加
   - クエリの改善

2. **メモリ使用量の削減**
   - 不要なデータのクリーンアップ
   - ストリーム処理の導入
   - 目標: アイドル時30-40MB

3. **UI応答性の向上**
   - 非同期処理の最適化
   - ローディング状態の改善
   - debounce/throttleの適用

---

### Phase 9-D: AI機能統合（オプション）

**推定時間**: 8-12時間

#### 機能詳細

1. **Ollama統合**
   - ローカルLLMの統合
   - 自然言語検索
   - ファイル内容の要約

2. **ベクトルデータベース統合**
   - ChromaDBまたはLanceDB
   - 意味検索
   - 類似ファイル検索

3. **タグ提案の高精度化**
   - AI/機械学習ベースの提案
   - コンテキスト理解
   - 提案精度の大幅向上
