## ⌨️ Phase 3: ホットキーテスト（3項目）

### 操作
1. ウィンドウが表示されている状態で **Ctrl+Shift+F** (Windows/Linux) または **Cmd+Shift+F** (macOS) を押す
2. もう一度同じキーを押す

### チェック項目
- [ ] 初回: ウィンドウが非表示になる
- [ ] 2回目: ウィンドウが再表示される
- [ ] 再表示時にフォーカスが当たる
